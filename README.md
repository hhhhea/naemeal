## 🎁 <내밀>의 의미
🌈 내 레시피를 남에게 흔쾌히 **내밀**다. <br>
🌈 나의 식사 (=> my meal [**내밀**]) <br>
🌈 맛있는 한 끼를 만들기 위한 나만의 **내밀**한 비법<br><br>

----
## 🎁 주요 기능
👉 로그인 / 회원가입(관리자/사용자) <br>
👉 프로필 등록 / 수정 <br>
👉 공지사항(관리자) <br>
👉 요리 프로그램 등록 / 수정 <br>
👉 요리 프로그램 참여 신청 <br>
👉 요리 프로그램 참여 신청 여부 조회 <br>
👉 나만의 레시피 등록 / 수정 / 조회 / 삭제 <br>
👉 댓글 <br><br>
